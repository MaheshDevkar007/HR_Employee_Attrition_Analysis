# HR Analytics Employee Attrition Analysis

## Project Overview
This project analyzes employee attrition using Python, Pandas, and NumPy.

## Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Analysis Performed
- Dataset Information
- Statistical Analysis
- Missing Values Check
- Department-wise Employee Count
- Department-wise Average Salary
- Attrition Analysis
- Gender Analysis
- Job Role Analysis
- Overtime Analysis
- Education Analysis
- Business Insights

## Dataset
IBM HR Analytics Employee Attrition Dataset
